## Check.sh

## Setup for zshrc, tmux.conf and vimrc
### via curl
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/mjonyh/config/master/symdotfiles.sh)"
```
### via wget
```
sh -c "$(wget https://raw.githubusercontent.com/mjonyh/config/master/symdotfiles.sh -0 -)"
```
